# NodeJs CURD operation using MongoDB 

Module used as below
* Express
* Mongoose
* Json Web Token(JWT)
* dotenv
* bcrypt


# API URL

Parent URL : /api/users

Add/ Register New users
/api/users/add

Login 
/api/users/login

Fetch all users 
/api/users/


Update existing users
/api/users/update

Delete users
/api/users/delete
